##  OWASP Top 10
OWASP Top 10 is a list of critical web application security risks.

##  Environment Setup

### OWASP Juice Shop
- Install node.js
- Run git clone https://github.com/juice-shop/juice-shop.git --depth 1 
- cd juice-shop
- Run npm install 
- Run npm start
- Browse to http://localhost:3000

### DVWA Installation
- Installed XAMPP
- Configured Apache & MySQL
- Deployed DVWA
- Set security level to LOW

OR : 

- git clone https://github.com/digininja/DVWA.git
- wget https://raw.githubusercontent.com/IamCarron/DVWA-Script/main/Install-DVWA.sh
- chmod +x Install-DVWA.sh
- sudo ./Install-DVWA.sh


### Burp Suite Configuration
- Enabled proxy (127.0.0.1:8080)
- Configured browser proxy
- Installed CA certificate

- Performed spidering
