# QWASP-TOP-10

# 🔐 Web Application Penetration Testing (OWASP Top 10)

## 📌 Project Overview
This project focuses on performing penetration testing on vulnerable web applications using the OWASP Top 10 framework. The objective is to identify, exploit, and provide remediation for common web vulnerabilities.

## 🎯 Objectives
- Understand OWASP Top 10 vulnerabilities
- Perform real-world web application attacks
- Document exploitation techniques
- Suggest remediation strategies

## 🛠️ Tools Used
- Burp Suite
- DVWA / OWASP Juice Shop
- Web Browser (Chrome/Firefox)

## 📅 Weekly Progress

### Week 1
- Studied OWASP Top 10
- Set up DVWA and QWASP Juice Shop
- Configured Burp Suite
- Performed reconnaissance and spidering

### Week 2
- SQL Injection
- Cross-Site Scripting (XSS)
- Broken Authentication
- Sensitive Data Exposure

### Week 3
- Security Misconfiguration
- Broken Access Control
- Created Proof of Concepts (PoC)

### Week 4
- Final Report Preparation
- Documentation and Review

## 📊 Key Outcomes
- Identified multiple vulnerabilities
- Successfully exploited web security flaws
- Provided remediation techniques

## 📌 Conclusion
This project enhanced practical knowledge of web security testing and ethical hacking techniques aligned with industry standards.
